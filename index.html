🔴 Early Game — Quality T1 + Speed T1

Early Game
Situação Early Game: Com Quality T1 Normal (1% de qualidade), qualquer módulo de Speed T1 no beacon (-0.5%/módulo) reduz rapidamente a qualidade. O ponto ótimo é 0 módulos no beacon para módulos normais. Só vale usar beacons com módulos Legendary ou Epic.

🟠 Mid Game — Quality T2 + Speed T2

Mid Game
Situação Mid Game: Com Quality T2 Normal (2% de qualidade), a penalidade do Speed T2 (-0.75%/módulo no beacon) zera a qualidade com apenas 2-3 módulos. Com módulos Legendary, o ponto ótimo sobe para 2 módulos no beacon.
